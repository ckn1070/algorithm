# 전화번호 목록

## 📌 문제 설명
전화번호부에 적힌 전화번호 중,  
**한 번호가 다른 번호의 접두어인 경우가 있는지** 확인하려고 합니다.

전화번호 목록이 담긴 배열 `phone_book`이 주어질 때,  
어떤 번호가 다른 번호의 접두어라면 `false`,  
그렇지 않으면 `true`를 반환하도록 `solution` 함수를 작성하세요.

---

## 📥 입력 조건
- `phone_book`: 전화번호가 담긴 배열  
  - 길이: 1 이상 1,000,000 이하
- 각 전화번호의 길이는 1 이상 20 이하
- 전화번호는 숫자로만 이루어진 문자열
- 중복된 전화번호는 없음

---

## 📤 출력 조건
- 접두어 관계가 존재하지 않으면 `true`, 존재하면 `false` 반환

---

## 🔍 입출력 예시

| phone_book | return |
|------------|--------|
| ["119", "97674223", "1195524421"] | false |
| ["123", "456", "789"] | true |
| ["12", "123", "1235", "567", "88"] | false |

---

## 🧠 해결 아이디어
- 전화번호를 **사전순(문자열)으로 정렬**합니다.
- 정렬된 상태에서는 **접두어 관계인 번호들이 반드시 인접**하게 위치합니다.
- 인접한 두 번호를 비교하여,
  - 앞 번호가 뒤 번호의 접두어인지 확인합니다.

---

## 🧪 주의 사항
- 모든 번호 쌍을 비교하면 시간 초과가 발생합니다.
- 문자열의 `startswith`(또는 슬라이싱) 비교를 활용하면 효율적입니다.
- 정렬 후 한 번의 선형 탐색으로 해결 가능합니다.
