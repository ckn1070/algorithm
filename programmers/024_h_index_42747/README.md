# H-Index

## 📌 문제 설명
과학자의 논문 인용 횟수를 나타내는 배열 `citations`가 주어집니다.  

H-Index란 다음과 같이 정의됩니다.

> 어떤 과학자가 발표한 논문 중  
> **h번 이상 인용된 논문이 h편 이상**이고,  
> 나머지 논문이 **h번 이하 인용**되었다면  
> 이때의 h가 해당 과학자의 H-Index입니다.

주어진 `citations` 배열을 기반으로  
과학자의 **H-Index**를 반환하도록 `solution` 함수를 작성하세요.

---

## 📥 입력 조건
- `citations`: 각 논문의 인용 횟수가 담긴 배열  
  - 길이: 1 이상 1,000 이하
  - 각 원소: 0 이상 10,000 이하의 정수

---

## 📤 출력 조건
- H-Index 값을 반환

---

## 🔍 입출력 예시

| citations | return |
|-----------|--------|
| [3, 0, 6, 1, 5] | 3 |

---

## 🧠 해결 아이디어
- H-Index는 **정렬 기반 문제**입니다.
- `citations`를 **내림차순 정렬**합니다.
- 정렬된 배열에서:
  - 인덱스 i(0-based)에 대해  
    `citations[i] >= i + 1` 이면 조건을 만족합니다.
- 이 조건을 만족하는 최대 `i + 1` 값이 H-Index입니다.

또는 오름차순 정렬 후,
- 전체 논문 수를 `n`이라 할 때  
  `citations[i] >= n - i` 인지 확인하는 방식도 가능합니다.

---

## 🧪 주의 사항
- 단순히 최대 인용 수가 H-Index는 아닙니다.
- 배열이 비어 있지 않으므로 최소값은 0 이상입니다.
- 입력 크기가 작아 **O(n log n)** 정렬 풀이로 충분합니다.
