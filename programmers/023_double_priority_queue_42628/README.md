# 이중우선순위큐

## 📌 문제 설명
이중 우선순위 큐는 다음 연산을 할 수 있는 자료구조입니다.

- `I 숫자` : 큐에 숫자를 삽입합니다.
- `D 1` : 큐에서 **최댓값**을 삭제합니다.
- `D -1` : 큐에서 **최솟값**을 삭제합니다.

연산을 모두 처리한 뒤,  
큐가 비어 있으면 `[0, 0]`을 반환하고,  
비어 있지 않으면 `[최댓값, 최솟값]`을 반환하도록  
`solution` 함수를 작성하세요.

---

## 📥 입력 조건
- `operations`: 연산을 나타내는 문자열 배열  
  - 길이: 1 이상 1,000,000 이하
  - 각 원소는 `"I 숫자"`, `"D 1"`, `"D -1"` 중 하나
- 삽입되는 숫자는 -10,000,000 이상 10,000,000 이하의 정수
- 삭제 연산 시 큐가 비어 있다면 해당 연산은 무시합니다.

---

## 📤 출력 조건
- 모든 연산을 수행한 뒤의 결과를 배열로 반환
  - 비어 있으면 `[0, 0]`
  - 비어 있지 않으면 `[최댓값, 최솟값]`

---

## 🔍 입출력 예시

| operations | return |
|------------|--------|
| ["I 16","D 1"] | [0,0] |
| ["I 7","I 5","I -5","D -1"] | [7,5] |

---

## 🧠 해결 아이디어
- **최솟값과 최댓값을 모두 빠르게 삭제/조회**해야 합니다.
- 일반적인 접근 방법:
  1. **최소 힙 + 최대 힙을 동시에 사용**
     - 삽입 시 두 힙 모두에 값을 넣습니다.
     - 삭제 시 한 힙에서 꺼낸 값을 다른 힙에서도 동기화 처리합니다.
  2. 또는 **정렬 가능한 자료구조(멀티셋 개념)**를 사용합니다.
- 파이썬의 경우 힙만으로는 동기화가 까다로워  
  **lazy deletion(지연 삭제)** 방식이 자주 사용됩니다.

---

## 🧪 주의 사항
- 큐가 비어 있을 때의 삭제 연산은 반드시 무시해야 합니다.
- 단순 리스트 정렬 방식은 연산 수가 많아 **시간 초과**가 발생할 수 있습니다.
- 최종 결과를 반환할 때는 반드시 **최댓값 먼저, 최솟값 다음** 순서입니다.
