# 주식가격

## 📌 문제 설명
초 단위로 기록된 주식가격이 담긴 배열 prices가 매개변수로 주어질 때,  
가격이 떨어지지 않은 기간은 몇 초인지를 return 하도록 solution 함수를 완성하세요.

---

## 📥 입력 조건
- `prices`의 각 가격은 1 이상 10,000 이하인 자연수입니다.
- `prices`의 길이는 2 이상 100,000 이하입니다.

---

## 📤 출력 조건
- 각 시점의 가격이 떨어지지 않은 기간(초)을 배열에 담아 반환합니다.

---

## 🔍 입출력 예시

| prices | return |
|---|---|
| [1, 2, 3, 2, 3] | [4, 3, 1, 1, 0] |

---

## 🧠 해결 아이디어
- 각 시점 i에 대해 이후 가격이 처음으로 떨어지는 시점을 찾습니다.
- 끝까지 떨어지지 않으면 마지막 시점까지의 시간을 계산합니다.
- 스택을 활용하면 O(n)으로 해결할 수 있습니다.

---

## 🧪 주의 사항
- 가격이 같은 경우는 “떨어진 것”이 아닙니다.
- 마지막 원소는 항상 0입니다.
- 입력 크기가 크므로 시간 복잡도에 주의해야 합니다.
