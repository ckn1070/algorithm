# 완주하지 못한 선수

## 📌 문제 설명
마라톤에 참여한 선수들의 이름이 담긴 배열 `participant`와  
완주한 선수들의 이름이 담긴 배열 `completion`이 주어집니다.

마라톤에 참여한 선수 중 **단 한 명만 완주하지 못했습니다**.  
완주하지 못한 선수의 이름을 반환하도록 solution 함수를 작성하세요.

---

## 📥 입력 조건
- `participant`: 마라톤에 참여한 선수들의 이름이 담긴 배열  
  - 길이: 1 이상 100,000 이하
- `completion`: 완주한 선수들의 이름이 담긴 배열  
  - 길이: `participant` 길이보다 1 작음
- 선수 이름은 소문자로 이루어져 있음
- **동명이인이 있을 수 있음**

---

## 📤 출력 조건
- 완주하지 못한 선수의 이름을 문자열로 반환

---

## 🔍 입출력 예시

| participant | completion | return |
|------------|------------|--------|
| ["leo","kiki","eden"] | ["eden","kiki"] | "leo" |
| ["marina","josipa","nikola","vinko","filipa"] | ["josipa","filipa","marina","nikola"] | "vinko" |
| ["mislav","stanko","mislav","ana"] | ["stanko","ana","mislav"] | "mislav" |

---

## 🧠 해결 아이디어
- 참가자 이름을 기준으로 **해시맵(딕셔너리)**을 사용해 인원 수를 카운트합니다.
- 완주자 명단을 순회하며 해당 이름의 카운트를 감소시킵니다.
- 카운트가 0이 되지 않은 이름이 **완주하지 못한 선수**입니다.

---

## 🧪 주의 사항
- 동명이인이 존재할 수 있으므로 단순 비교는 불가능합니다.
- 입력 크기가 크기 때문에 시간 복잡도는 **O(n)** 수준이어야 합니다.
