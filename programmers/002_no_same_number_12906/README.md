# 같은 숫자는 싫어

## 📌 문제 설명
배열 `arr`가 주어집니다.  
배열 `arr`의 각 원소는 숫자 0부터 9까지로 이루어져 있습니다.

이 배열에서 **연속적으로 나타나는 숫자는 하나만 남기고 전부 제거**하려고 합니다.  
단, 제거된 후 남은 숫자들의 **순서는 유지**해야 합니다.

---

## 📥 입력 조건
- `arr`: 숫자 0부터 9까지로 이루어진 배열
  - 길이: 1 이상 1,000,000 이하

---

## 📤 출력 조건
- 연속 중복이 제거된 배열을 반환

---

## 🔍 입출력 예시

| arr | return |
|----|--------|
| [1,1,3,3,0,1,1] | [1,3,0,1] |
| [4,4,4,3,3] | [4,3] |

---

## 🧠 해결 아이디어
- **스택(또는 결과 배열)**을 하나 생성합니다.
- 배열을 앞에서부터 순회하면서  
  - 스택이 비어 있거나,  
  - 스택의 마지막 값과 현재 값이 다를 경우에만 push 합니다.
- 이렇게 하면 연속된 중복 숫자가 자동으로 제거됩니다.

---

## 🧪 주의 사항
- 단순 중복 제거가 아니라 **연속 중복만 제거**해야 합니다.
- 입력 배열의 길이가 매우 크므로 **O(n)** 시간 복잡도로 해결해야 합니다.
