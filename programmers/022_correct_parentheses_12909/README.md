# 올바른 괄호

## 📌 문제 설명
괄호가 바르게 짝지어졌다는 것은 `'('` 문자로 열렸으면 반드시 짝지어서 `')'` 문자로 닫혀야 한다는 뜻입니다.

예를 들어
- `"()()"` 또는 `"(())()"` 는 올바른 괄호입니다.
- `")()("` 또는 `"(()("` 는 올바르지 않은 괄호입니다.

괄호 문자열 `s`가 주어졌을 때,  
이 문자열이 **올바른 괄호**이면 `true`, 아니면 `false`를 반환하도록  
`solution` 함수를 작성하세요.

---

## 📥 입력 조건
- `s`: `'('` 또는 `')'` 로만 이루어진 문자열
  - 길이: 1 이상 100,000 이하

---

## 📤 출력 조건
- 올바른 괄호이면 `true`, 아니면 `false`를 반환

---

## 🔍 입출력 예시

| s | return |
|---|--------|
| "()()" | true |
| "(())()" | true |
| ")()(" | false |
| "(()(" | false |

---

## 🧠 해결 아이디어
- 스택(또는 카운터)을 사용합니다.
- 왼쪽부터 순회하며:
  - `'('`이면 스택에 push(또는 count 증가)
  - `')'`이면 스택에서 pop(또는 count 감소)
- 순회 중에 `')'`가 나왔는데 스택이 비어 있으면 올바르지 않습니다.
- 순회가 끝난 뒤 스택이 비어 있으면 올바른 괄호입니다.

---

## 🧪 주의 사항
- 시작이 `')'`인 경우 즉시 `false`입니다.
- 문자열을 전부 처리한 뒤에도 `'('`가 남아 있으면 `false`입니다.
- 입력 크기가 크므로 **O(n)** 으로 처리해야 합니다.
