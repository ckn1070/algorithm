# 디스크 컨트롤러

## 📌 문제 설명
하드디스크는 한 번에 하나의 작업만 수행할 수 있습니다.  
각 작업은 요청 시점과 작업 소요 시간이 주어집니다.

디스크 컨트롤러는 **작업의 요청부터 종료까지 걸린 시간의 평균**을 최소화하려고 합니다.  
이때의 평균 시간을 반환하도록 `solution` 함수를 작성하세요.

각 작업은 `[요청 시점, 소요 시간]` 형태로 주어집니다.

---

## 📥 입력 조건
- `jobs`: 작업 정보가 담긴 2차원 배열  
  - 길이: 1 이상 500 이하
  - 각 원소: `[requestTime, duration]`
  - `requestTime`: 0 이상 1,000 이하
  - `duration`: 1 이상 1,000 이하

---

## 📤 출력 조건
- 작업 요청부터 종료까지 걸린 시간의 **평균값의 정수 부분**을 반환

---

## 🔍 입출력 예시

| jobs | return |
|------|--------|
| [[0, 3], [1, 9], [2, 6]] | 9 |

---

## 🧠 해결 아이디어
- 평균 시간을 최소화하려면 **현재 시점에서 처리 가능한 작업 중 소요 시간이 가장 짧은 작업**을 먼저 수행해야 합니다.
- 이를 위해 **우선순위 큐(최소 힙)**를 사용합니다.
- 처리 절차:
  1. 요청 시점을 기준으로 작업을 정렬합니다.
  2. 현재 시점까지 요청된 작업을 힙에 넣습니다.
  3. 힙에서 소요 시간이 가장 짧은 작업을 꺼내 처리합니다.
  4. 현재 시점을 갱신하고, 각 작업의 대기 시간을 누적합니다.
- 모든 작업이 처리될 때까지 위 과정을 반복합니다.

---

## 🧪 주의 사항
- 처리할 수 있는 작업이 없을 경우, **현재 시점을 다음 작업의 요청 시점으로 이동**해야 합니다.
- 평균값 계산 시 **소수점 이하는 버림**합니다.
- 작업 수가 많지 않지만, 그리디 + 힙 구조를 정확히 구현해야 합니다.
