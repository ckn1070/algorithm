# 전력망을 둘로 나누기

## 📌 문제 설명
n개의 송전탑이 전선을 통해 하나의 트리 형태로 연결되어 있습니다.  
당신은 이 전선들 중 하나를 끊어서 현재의 전력망 네트워크를 2개로 분할하려고 합니다.  
이때, 두 전력망이 갖게 되는 송전탑의 개수를 최대한 비슷하게 맞추고자 합니다.

송전탑의 개수 `n`, 그리고 전선 정보 `wires`가 매개변수로 주어집니다.  
전선들 중 하나를 끊어서 송전탑 개수가 가능한 비슷하도록 두 전력망으로 나누었을 때,  
두 전력망이 가지고 있는 송전탑 개수의 차이(절대값)를 `return` 하도록 `solution` 함수를 완성해주세요.

---

## 📥 입력 조건
- `n`은 2 이상 100 이하인 자연수입니다.
- `wires`는 길이가 `n-1`인 정수형 2차원 배열입니다.
- `wires`의 각 원소는 `[v1, v2]` 2개의 자연수로 이루어져 있으며, 이는 전력망의 `v1`번 송전탑과 `v2`번 송전탑이 전선으로 연결되어 있다는 것을 의미합니다.
- `1 ≤ v1 < v2 ≤ n` 입니다.
- 전력망 네트워크가 하나의 트리 형태가 아닌 경우는 입력으로 주어지지 않습니다.

---

## 📤 출력 조건
- 전선 하나를 끊어 두 전력망으로 분할했을 때,
  두 전력망의 송전탑 개수 차이(절대값)의 **최솟값**을 반환합니다.

---

## 🔍 입출력 예시

| n | wires | result |
|---:|---|---:|
| 9 | [[1,3],[2,3],[3,4],[4,5],[4,6],[4,7],[7,8],[7,9]] | 3 |
| 4 | [[1,2],[2,3],[3,4]] | 0 |
| 7 | [[1,2],[2,7],[3,7],[3,4],[4,5],[6,7]] | 1 |

---

## 🧠 해결 아이디어
- 트리에서 간선(전선) 하나를 제거하면 그래프가 정확히 2개의 연결 컴포넌트로 나뉩니다.
- 모든 전선 `[a, b]`를 하나씩 “끊어본다”라고 가정하고,
  - 해당 간선을 무시한 상태에서 BFS/DFS로 한 쪽 컴포넌트의 노드 수를 셉니다.
  - 다른 쪽은 `n - count`이므로,
  - 차이는 `abs(n - 2*count)`로 계산할 수 있습니다.
- 모든 전선에 대해 위 과정을 반복하며 최솟값을 갱신합니다.
- n ≤ 100이므로, 간선마다 BFS/DFS를 돌려도 충분합니다.

---

## 🧪 주의 사항
- 반드시 “전선 하나만” 끊습니다(동시에 여러 개를 끊는 경우는 없음).
- `wires`는 트리를 이루므로, 간선을 하나 제거하면 정확히 2개로 분리됩니다.
- 방문 배열(visited)을 매 시도마다 초기화해야 합니다.
