# 모의고사

## 📌 문제 설명
수포자(수학 포기자) 삼인방은 모의고사에 수학 문제를 전부 찍으려 합니다.  
각 학생은 일정한 패턴으로 답을 찍습니다.

- 1번 수포자: `1, 2, 3, 4, 5` 반복  
- 2번 수포자: `2, 1, 2, 3, 2, 4, 2, 5` 반복  
- 3번 수포자: `3, 3, 1, 1, 2, 2, 4, 4, 5, 5` 반복  

문제의 정답이 담긴 배열 `answers`가 주어질 때,  
가장 많은 문제를 맞힌 사람(들)의 번호를 배열에 담아 반환하도록 `solution` 함수를 작성하세요.

---

## 📥 입력 조건
- `answers`: 1번 문제부터 마지막 문제까지의 정답이 담긴 배열  
  - 길이: 1 이상 10,000 이하
  - 각 원소: 1 이상 5 이하의 정수

---

## 📤 출력 조건
- 가장 높은 점수를 받은 사람의 번호를 오름차순으로 담은 배열을 반환

---

## 🔍 입출력 예시

| answers | return |
|--------|--------|
| [1,2,3,4,5] | [1] |
| [1,3,2,4,2] | [1,2,3] |

---

## 🧠 해결 아이디어
- 각 수포자의 찍기 패턴을 배열로 정의합니다.
- `answers`를 순회하며 각 수포자의 패턴과 비교하여 맞힌 개수를 카운트합니다.
  - 패턴은 반복되므로 인덱스를 `i % pattern.length`로 접근합니다.
- 세 사람의 점수 중 최댓값을 구한 뒤,
  최댓값과 같은 점수를 가진 사람의 번호를 결과 배열에 담습니다.

---

## 🧪 주의 사항
- 패턴 길이가 서로 다르므로 modulo 연산을 정확히 적용해야 합니다.
- 동점자가 있을 수 있으므로 오름차순으로 모두 반환해야 합니다.
