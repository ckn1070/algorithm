# 폰켓몬

## 📌 문제 설명
당신은 N마리의 폰켓몬 중에서 **N/2마리**를 가져가도 좋다는 허락을 받았습니다.  
폰켓몬은 종류가 여러 가지이며, 같은 종류의 폰켓몬이 여러 마리 있을 수 있습니다.

가질 수 있는 폰켓몬의 수는 **최대 N/2마리**이지만,  
종류는 최대한 다양하게 가져가고 싶습니다.

배열 `nums`가 주어질 때,  
가질 수 있는 폰켓몬 종류 수의 **최댓값**을 반환하도록 `solution` 함수를 작성하세요.

---

## 📥 입력 조건
- `nums`: 폰켓몬의 종류 번호가 담긴 배열  
  - 길이: 2 이상 10,000 이하 (항상 짝수)
  - 각 원소는 1 이상 200,000 이하의 자연수

---

## 📤 출력 조건
- 선택할 수 있는 폰켓몬 종류 수의 최댓값을 반환

---

## 🔍 입출력 예시

| nums | return |
|------|--------|
| [3,1,2,3] | 2 |
| [3,3,3,2,2,4] | 3 |
| [3,3,3,2,2,2] | 2 |

---

## 🧠 해결 아이디어
- 최대로 다양한 종류를 고르려면 **중복을 제거한 종류 개수**를 세면 됩니다.
- 실제로 가져갈 수 있는 마리 수는 `N/2`이므로,
  - `종류 개수`와 `N/2` 중 **더 작은 값**이 정답입니다.

---

## 🧪 주의 사항
- 종류가 아무리 많아도 가져갈 수 있는 마리 수는 `N/2`를 넘을 수 없습니다.
- `set` 등을 이용해 중복 제거 후 종류 개수를 계산하면 간단합니다.
