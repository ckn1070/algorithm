# K번째수

## 📌 문제 설명
배열 `array`의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때,  
k번째에 있는 수를 구하려 합니다.

예를 들어 `array = [1, 5, 2, 6, 3, 7, 4]`,  
`commands = [[2, 5, 3], [4, 4, 1], [1, 7, 3]]` 라면

- [2, 5, 3] → array의 2~5번째: [5,2,6,3] → 정렬: [2,3,5,6] → 3번째: 5  
- [4, 4, 1] → array의 4~4번째: [6] → 정렬: [6] → 1번째: 6  
- [1, 7, 3] → array의 1~7번째: [1,5,2,6,3,7,4] → 정렬: [1,2,3,4,5,6,7] → 3번째: 3  

각 명령에 대한 결과를 배열로 반환하세요.

---

## 📥 입력 조건
- `array`: 1 이상 100 이하의 자연수로 이루어진 배열  
  - 길이: 1 이상 100 이하
- `commands`: `[i, j, k]` 형태의 2차원 배열  
  - 길이: 1 이상 50 이하
- i, j, k는 모두 1 이상이며 i ≤ j

---

## 📤 출력 조건
- `commands`의 각 원소에 대해 계산한 결과를 담은 배열을 반환

---

## 🔍 입출력 예시

| array | commands | return |
|------|----------|--------|
| [1, 5, 2, 6, 3, 7, 4] | [[2,5,3],[4,4,1],[1,7,3]] | [5, 6, 3] |

---

## 🧠 해결 아이디어
- 각 `command`마다:
  1) `array`를 i~j 구간으로 **슬라이싱**
  2) 슬라이스한 배열을 **정렬**
  3) 정렬된 배열의 **k번째 원소**를 선택
- 각 결과를 모아 반환합니다.

---

## 🧪 주의 사항
- i, j, k는 **1-based 인덱스**이므로 슬라이싱/접근 시 0-based로 변환이 필요합니다.
- 입력 크기가 작아 단순 정렬 방식으로도 충분합니다.
