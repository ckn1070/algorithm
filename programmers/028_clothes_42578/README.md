# 의상

## 📌 문제 설명
코니는 매일 다른 옷을 조합하여 입는 것을 좋아합니다.  
예를 들어 코니가 가진 옷이 아래와 같고 오늘 코니가 동그란 안경, 긴 코트를 입었다면  
다음날은 청바지를 추가로 입거나 동그란 안경 대신 검정 선글라스를 착용하거나 해야 합니다.

- 종류: 얼굴 / 이름: 동그란 안경, 검정 선글라스
- 종류: 상의 / 이름: 파란색 티셔츠
- 종류: 하의 / 이름: 청바지
- 종류: 겉옷 / 이름: 긴 코트

코니는 각 종류별로 **최대 1가지 의상만 착용**할 수 있으며,  
오늘 입은 옷의 조합과 **서로 다른 조합**의 옷을 입어야 합니다.

코니가 가진 의상들이 담긴 배열 `clothes`가 주어질 때,  
코니가 입을 수 있는 옷의 조합의 수를 반환하도록 `solution` 함수를 작성하세요.

---

## 📥 입력 조건
- `clothes`: 의상의 이름과 종류가 담긴 2차원 배열  
  - 길이: 1 이상 30 이하
  - 각 원소는 `[의상 이름, 의상 종류]` 형태
  - 의상 이름은 중복되지 않음
  - 의상 종류는 문자열

---

## 📤 출력 조건
- 서로 다른 옷의 조합의 수를 반환

---

## 🔍 입출력 예시

| clothes | return |
|--------|--------|
| [["yellow_hat","headgear"],["blue_sunglasses","eyewear"],["green_turban","headgear"]] | 5 |
| [["crow_mask","face"],["blue_sunglasses","face"],["smoky_makeup","face"]] | 3 |

---

## 🧠 해결 아이디어
- 의상 종류별로 개수를 **해시맵**으로 카운트합니다.
- 각 종류마다:
  - 해당 종류의 옷을 입는 경우 + 입지 않는 경우(1)를 고려합니다.
- 모든 종류의 경우의 수를 곱한 뒤,
  **아무것도 입지 않는 경우 1가지를 제외**합니다.

---

## 🧪 주의 사항
- 최소 한 가지 이상의 옷은 반드시 착용해야 합니다.
- 종류별로 하나만 선택 가능하므로 **곱셈 원리**를 적용합니다.
