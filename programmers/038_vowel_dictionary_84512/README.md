# 모음사전

## 📌 문제 설명
사전에 알파벳 모음 'A', 'E', 'I', 'O', 'U'만을 사용하여 만들 수 있는, 길이 5 이하의 모든 단어가 수록되어 있습니다. 사전에서 첫 번째 단어는 "A"이고, 그다음은 "AA"이며, 마지막 단어는 "UUUUU"입니다.

단어 하나 word가 매개변수로 주어질 때, 이 단어가 사전에서 몇 번째 단어인지 return 하도록 solution 함수를 완성해주세요.

---

## 📥 입력 조건
- `word`의 길이는 1 이상 5 이하입니다.
- `word`는 알파벳 대문자 'A', 'E', 'I', 'O', 'U'로만 이루어져 있습니다.

---

## 📤 출력 조건
- `word`가 사전에서 몇 번째 단어인지(1부터 시작) 정수로 반환합니다.

---

## 🔍 입출력 예시

| word | result |
|---|---:|
| "AAAAE" | 6 |
| "AAAE" | 10 |
| "I" | 1563 |
| "EIO" | 1189 |

---

## 🧠 해결 아이디어
- 길이 1~5, 문자 5개이므로 가능한 모든 단어 수는 3905개입니다.
- DFS(완전탐색)로 모든 단어를 사전 순서대로 생성할 수 있습니다.
- 생성 순서가 곧 사전 순서이므로, 해당 단어가 등장하는 위치를 찾으면 됩니다.
- 또는 자리별 가중치를 이용한 수학적 계산으로도 구할 수 있습니다.

---

## 🧪 주의 사항
- 사전 순서는 길이가 짧은 단어도 포함됩니다. (A → AA → AAA …)
- 탐색 중 target 단어를 찾으면 즉시 종료할 수 있습니다.

---
